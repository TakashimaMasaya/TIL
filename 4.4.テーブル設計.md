**要件**
- ログインログアウト機能がある
- ユーザーはニックネームとアバター画像を登録できる
- ユーザーは質問を投稿できる
- ユーザーは自分の質問を編集・削除できる
- ユーザーは質問に対して回答ができる
- ユーザーは質問を検索できる
- 質問があった際に全員に対して質問があった旨をメールで通知する（ただし自分は除く）
- 質問に対して回答があった場合は質問者および当該質問に回答したユーザーに対してメールで通知する。（ただし自分は除く）
- 質問はページングできる
- 管理画面がある
- 管理画面へは権限を付与されたユーザーしか入れない
- 管理画面では全てのリソースを削除できる

**users**
```
  id: int
  name: string
  email: string
  password: string
  password_digest: string
  admin: boolean
```
**question**
```
  id: int
  title: string
  description: text
  user_id: int
```

**answer**
```
  id: int
  title: string
  description: text
  user_id: int
  question_id: int
```

これプラス`active_storage:install`で自動生成される`active_storage_blobs`と`active_storage_attachments`の２つのテーブルです。
